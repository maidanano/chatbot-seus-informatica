# Diccionario de datos

Archivo principal: `datos/ordenes.txt`

Campos:

- `orden`: identificador de la orden.
- `cliente`: nombre del cliente.
- `telefono`: número de contacto.
- `equipo`: equipo ingresado.
- `falla`: descripción breve del problema.
- `estado`: estado actual de la orden.
- `tecnico_asignado`: área responsable.
